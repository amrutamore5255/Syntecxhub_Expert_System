# Syntecxhub_Expert_System
A Rule-Based Expert System is an AI project that uses predefined if-then rules to make decisions based on user inputs or facts. The system applies logical reasoning and forward chaining to infer conclusions, supports multi-step reasoning, and displays the reasoning path for better understanding.

Features
Accepts user inputs/symptoms
Uses if-then rule logic
Implements forward chaining
Supports multi-step reasoning
Displays inference steps
Beginner-friendly AI project

Technologies Used
Python
Lists & Dictionaries
Conditional Statements
AI Rule-Based Concepts
Example

Input:

Fever
Cough
Headache

Output:

Possible Flu Detected

Applications
Medical Diagnosis Systems
Customer Support Systems
Troubleshooting Systems
Recommendation Systems

Objective

The objective of this project is to understand the basics of Artificial Intelligence, knowledge representation, and logical decision-making using rule-based systems.
